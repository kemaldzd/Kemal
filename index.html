Kenapa STEM?

Saya percaya bahawa STEM adalah masa depan. Daripada Kecerdasan Buatan sehinggalah kepada perisian komputer, semuanya bermula dengan keberanian untuk belajar. Saya bersedia untuk mencabar diri, mengejar semula pelajaran yang tertinggal dan berusaha lebih untuk membuktikan bahawa saya layak berada dalam bidang ini.

Saya juga sedar bahawa saya telah membuat kesilapan sebelum ini kerana tidak memilih ASK lebih awal. Tetapi kini, saya sedar bahawa ASK sebenarnya adalah jiwa saya. Sejak sekolah rendah lagi, saya sudah menunjukkan minat terhadap pengaturcaraan, terutamanya bahasa JavaScript. Saya pernah mencipta animasi sendiri, tetapi tidak sempat untuk render kerana spesifikasi laptop saya tidak menyokong. Namun begitu, ia tidak mematahkan semangat saya untuk terus belajar.
